Carga del programa

Nada más iniciar el programa, aparecerá con el siguiente aspecto:

En la parte superior aparece el menú principal.  A la izquierda está el árbol del proyecto y a la derecha el editor de archivos, inicialmente vacío puesto que no se ha cargado todavía ninguno.

Nota: Pulsando la tecla F10 en cualquier momento hará que el menú principal aparezca o desaparezca, independientemente de la configuración.

Carga del directorio de proyecto

Para poder trabajar hay que cargar un directorio de proyecto.  Se puede hacer de dos formas:

  • Seleccionando en el menú principal la opción Proyecto → Abrir....
  • Pulsando con el botón secundario del ratón sobre el árbol de proyecto para abrir el menú contextual y seleccionando la opción Abrir....

En cualquiera de los dos casos se abrirá un diálogo que permitirá seleccionar un directorio de disco.  Una vez seleccionado, la estructura y los archivos contenidos se cargarán y se mostrarán en el árbol de proyecto.

Nota: Es posible que, por defecto, no se muestren todos los archivos y directorios contenidos en el directorio del proyecto.  Esto se debe a que la configuracón por defecto establece que sólo se carguen los archivos reconocidos como fuentes o textos, y a una profundidad determinada.  Puede cambiarse este comportamiento en la Configuración.

Puede desplegar o replegar los directorios para mostrar su contenido pulsando con el ratón sobre el icono que se encuentra al lado de los directorios (carpetas).  También puede cambiar el tamaño del área reservada al árbol de proyecto pulsando y arrastrando en el margen derecho de dicho área.

Carga de archivos

Para cargar un archivo, basta con hacer una doble pulsación sobre él en el árbol de proyecto.  Puede abrir varios archivos, cada uno de los cuales aparecerá en una lengüeta diferente.  Al seleccionar un archivo que ya haya sido abierto se seleccionará la lengüeta del mismo automáticamente.

Si el lenguaje de programación es reconocido, se mostrará con resaltado de sintáxis.

Editar archivos

Por ahora, el programa permite una edición básica, con combinaciones de tecla tipo IBM CUA (Ctrl-Insert, Shift-Insert, etc.) y Xerox PARC (Ctrl-C, Ctrl-V, Ctrl-Z, etc.).  También permite la selección de texto con el ratón o manteniendo la tecla de mayúsculas pulsada mientras se usan los cursores para mover el cursor.

El editor incluye, también, una herramienta de autocompletado.  Pulse la combinación Ctrl+P en cualquier momento para activarla; aparecerá una lista de palabras.  Según vaya escribiendo esta lista irá cambiando mostrando coincidencias.  Puede usar los cursores para seleccionar una de las palabra y pulse la barra espaciadora para insertar la palabra seleccionada.  El editor irá añadiendo palabras a la lista según vaya editando el archivo.

Importante: El editor usa UTF-8 para la codificación de caracteres.  Esto puede provocar algunos problemas en ciertos casos (por ejemplo, no reconocer correctamete las palabras si estas tienen acentos).  Se pretende que esté solucionado para la versión 1.0 (recuerde que la versión actual está todavía en desarrollo).

Guardar archivos

Existen varias formas de guardar un archivo editado.

  • Pulsando sobre el icono del disquete en la barra de tareas.
  • Seleccionando en el menú principal la opción Archivo → Guardar.
  • Pulsando con el botón secundario del ratón sobre el editor de archivos para abrir el menú contextual y seleccionando la opción Guardar.

Otras operaciones

Pueden realizarse más operaciones, tales como añadir archivos, cambiar el nombre de los mismos, etc.  Para ello hay que usar los menús contextuales que aparecen pulsando con el botón secundario del ratón sobre el árbol de proyecto o sobre el editor de archivos.

Así mismo, en el diálogo de Configuración puede modificarse algunos comportamientos del programa.

Por último, recordar que la presente versión está todavía en pleno desarrollo y carece de muchas de las opciones y funciones planificadas.
