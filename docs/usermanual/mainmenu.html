MLSDE

Options related with the application.

Configuration...

Opens the configuration dialog.

About...

Opens the About dialog box.

This shows information about the application, as the current version, authors and license.

Quit

Quits the application.  This has the same effect than closing the window.

Project

Options related with projects.

Open...

Opens a new project.

This will show a dialog that allows to select the directory where the project is stored.
